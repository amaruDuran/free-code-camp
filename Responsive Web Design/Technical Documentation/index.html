Listas

Las listas son una de las estructuras de datos más utilizadas en el mundo. Estas permiten multiples tipos de datos dentro de la misma. Dado que python no es fuertemente tipado. Se pueden Guardar desde números, strings, hasta objetos (tipos de datos abstractos - TDAs).

Eliminar Elemento

list.pop([i])

Complejidad Temporal: O(1) sin parámetros, O(n) con parametros

Le pasamos la posición i en la que se encuentra el elemento, lo elimina de la lista y lo devuelve.
Podemos no pasarle el parámetro i, en ese caso, la función pop elimina el último elemento de la lista y lo devuelve.
